# SymPy Tutorial


## Instructions

1. Install SymPy:
```bash
pip install sympy
```
2. Clone the repository:

```bash
git clone https://github.com/juliaj25/Project4.git
cd Project4
```
3. Fill in code to solve problem:

```bash
code pendulum_problem.py
```
4. After you've completed filling in the code, compare your result to the solution:
```bash
python pendulum_solution.py
```
* If you need to reference demo code:
```bash
code demo.py
```
